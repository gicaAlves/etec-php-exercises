## Running

I'm using [XAMPP](https://www.apachefriends.org/) to run the local server.

- Clone this repository under `<xampp_dir>/htdocs/`
- Start the Apache Server using XAMPP
- Go to `http://localhost/exercises/<exercise_name>`
